# BYOK — RBAC

## Роль
- `user` — управляет только своим ключом.

## Правила
- Все `/v1/byok/*` — `userId` == `sub` JWT.
- Plaintext ключ не возвращается клиенту ни при каких условиях.
- `get_plaintext_key` — внутренний вызов только Chat Orchestrator, не через HTTP.
- Нельзя получить/удалить/переключить чужой ключ.
