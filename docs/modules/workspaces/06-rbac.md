# Workspaces — RBAC

- Роль `user`. Все операции (CRUD workspace, файлы, привязка чатов) ограничены ресурсами `sub`.
- Чужой workspace → `404`. Привязка чужого `attachmentId` → `403`/`404`.
- Нет admin-операций.
- `instructions` — пользовательский текст, не секрет; но логирование тела привязано к общим redaction-правилам (без утечки содержимого файлов).
