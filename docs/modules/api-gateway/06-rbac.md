# API Gateway — RBAC

## Роли
- `user` — единственная роль. Аутентифицируется JWT.

## Правила доступа
- Все `/v1/*` требуют валидный JWT.
- Доступ только к собственным ресурсам: `userId` тела == `sub` токена (`403` иначе).
- `/health`, `/healthz` (алиас `/health`), `/ready` — без auth, публичные.
- `/metrics` — без user-JWT, защищён сетевой политикой или scrape-токеном.

## Нет admin
Admin-эндпоинтов на старте нет. При появлении — отдельный ADR и роль `admin` с собственным issuer/scope.
